===========
Time Period
===========

Python module for determining if a datetime is within a time
period.

Based on the `original TimePeriod module`_ written by Paul Boyd <boyd.paul2@gmail.com>.


Installation
============

.. code-block:: bash

    $ sudo pip install timeperiod2
